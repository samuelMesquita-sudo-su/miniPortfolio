Algumas game engines

O que é uma game engine?
é um software de desenvolvimento que oferece um conjunto de ferramentas e funcionalidades prontas para simplificar e acelerar a criação de jogos eletrônicos

Game Engines

Unity

é uma plataforma de desenvolvimento 3D e em tempo real, usada principalmente para criar jogos

Unreal Engine

é uma poderosa plataforma de criação 3D em tempo real da Epic Games
